◆ 1. 서비스 개요

본 웹앱은 동서대학교 ESG 굿아이디어 공모전을 위해 제작한 시범용(Prototype) 버전입니다.
오늘의 미션 추천, 카테고리별 ESG 실천, 경험치/레벨 시스템, 뱃지 시스템, 누적 차트, 히스토리 기능 등을 포함합니다.


◆ 2. 주요 기능

- ESG 미션 수행
- 환경(E), 존중(S), 소통(G) 카테고리별 총 9개 미션 제공
- 미션 클릭 시 경험치 +10
- 하루 3개의 서로 다른 미션 완료 시 “오늘 목표 달성”

레벨 & 경험치
- 10 포인트 = 1 레벨업
- 진행도 바 실시간 반영
- 레벨업 시 마스코트 애니메이션 + 폭죽 효과

뱃지 시스템
- 가장 많이 수행한 카테고리 → 대표 뱃지 1개
- 레벨마다 메달 추가

ESG 기여도 차트
- 오늘 수행한 미션 기준
- 환경/존중/소통 비율을 도넛차트로 시각화

히스토리 기능
- 날짜별 수행한 ESG 미션 전체 확인 가능

안내 팝업
- 시범 버전 안내 포함


◆ 3. 브라우저 호환성 관련 이슈
🔸 문제
Microsoft Edge 브라우저에서
경험치 증가가 정상적으로 동작하지 않는 현상 발생 가능

🔸 원인 추정
Edge의 localStorage 처리 타이밍/렌더링 충돌
Script 실행 순서가 Chrome·Whale과 다르게 작동하여 값이 초기화되거나 반영 누락
모바일 WebView 기반 Edge 버전에서 더 자주 발생

🔸 해결 여부
코드 상 문제는 해결됨
하지만 Edge는 동일 코드에서도 저장/반영 실패하는 경우가 있어
사용자 측 브라우저 호환성 문제로 판단됩니다.


◆ 4. 권장 브라우저
다음 브라우저에서 정상 작동 확인됨:
✔ 네이버 웨일(Whale)
✔ 구글 크롬(Google Chrome)
✔ 모바일 웨일
✔ 크롬 기반 브라우저들 대부분

⚠ Microsoft Edge 사용 시 경험치 반영 문제가 발생할 수 있으니 비권장



◆ NAN(숫자 아님) 표시 관련 안내
일부 환경에서 경험치(EXP) 또는 레벨(Level)이
NaN으로 표시되는 경우가 드물게 발생할 수 있습니다.

🔸 오류가 나타나는 조건
로컬스토리지(LocalStorage)에 저장된 points 값이
정상적인 숫자 형태로 불러와지지 못했을 때
특히, Microsoft Edge 브라우저 사용 시 더 자주 발생

🔸 이유
Edge 브라우저는 localStorage 값을 렌더링 전에 읽거나
비동기 처리 타이밍이 Chrome/Whale과 달라,
숫자를 불러오기 전에 null → NaN으로 변환되는 사례가 존재함.

◆ NAN 발생 시 해결 방법
NAN이 표시될 경우 브라우저 콘솔에서 초기화 명령을 입력하면 즉시 정상화됩니다.

✔ 콘솔 명령어
localStorage.removeItem("points");
localStorage.removeItem("history");
→ 새로고침하면 레벨과 경험치가 정상 숫자로 돌아옵니다.

(데이터 초기화 원치 않을 경우 removeItem("points") 만 사용 가능)
