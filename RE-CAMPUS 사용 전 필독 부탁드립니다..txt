1. 서비스 개요
본 웹앱은 동서대학교 ESG 굿아이디어 공모전 제출을 위해 제작한 시범용 웹앱(Prototype) 입니다.
주요 구성 요소는 다음과 같습니다.

- ESG 미션 수행 & 카테고리 필터 (환경·존중·소통)
- 하루 3개 미션 달성 시스템
- 경험치 & 레벨업 시스템
- 뱃지 시스템
- 히스토리(Log) 기능
- ESG 기여도 시각화(도넛 차트)
- 초기 안내 팝업



2. 주요 기능 요약
✔ ESG 미션
- 총 9개 미션 제공
- 미션 수행 시 경험치 +10
- 하루 3개의 서로 다른 미션 완료 시 “오늘 목표 달성”

✔ 레벨 & 경험치
- 10 포인트 = 1 레벨
- 레벨업 시 마스코트 애니메이션 + 폭죽 효과

✔ 대표 뱃지 & 메달
- 가장 많이 수행한 카테고리에 따라 대표 뱃지 1종 부여
- 레벨업마다 메달 추가

✔ ESG 도넛 차트
- 오늘 수행한 미션 수(E/S/G) 기준으로 시각화

✔ 히스토리 기능
- 날짜별 수행 미션 전체를 확인 가능



3. 브라우저 호환성 안내 (중요)
일부 브라우저에서 로컬 데이터(localStorage) 반영 시점이 달라
경험치 업데이트가 정상적으로 작동하지 않을 수 있습니다.

🚫 Microsoft Edge 브라우저에서 발생한 문제
- 경험치 증가가 적용되지 않음
- 레벨 바가 갱신되지 않는 현상
- localStorage 값이 NaN으로 표시되는 경우 발생

📌 원인
Edge의 비동기 렌더링 타이밍 문제
localStorage 값이 렌더 전에 읽혀 null → NaN으로 변환되는 사례 존재
크롬 기반이지만 WebView 계열은 캐시/저장 처리 방식이 상이함



4. 권장 브라우저
아래 환경에서 정상 작동 확인됨:

✔ 네이버 웨일(Whale)
✔ 구글 크롬(Google Chrome)
✔ 모바일 환경

⚠ Microsoft Edge는 비권장
(경험치·레벨 반영 오류 위험 있음)



5. 캐싱 이슈 해결 안내 (v 파라미터)
웨일/크롬에서 간혹 JS 파일 캐시가 남아
업데이트가 반영되지 않는 사례가 있습니다.

✔ 해결 방법
다음과 같이 버전 파라미터를 붙여 접속하면 정상 작동합니다.
👉 https://2025-dsu-esg.vercel.app/?v=2

오작동 시에는
?v=3, ?v=4 등 숫자를 바꿔 접속하면 최신 코드가 로드됩니다.



6. NAN(숫자 아님) 표시 오류 안내
일부 환경에서 레벨 또는 경험치가 NaN으로 표시될 수 있음.

✔ 주요 원인
- localStorage 값이 숫자로 로드되기 전 페이지 렌더링 시작
- Edge에서 더 자주 발생하는 현상

✔ 해결 방법 (데이터 초기화)
브라우저 콘솔에서 아래 명령 실행:

localStorage.removeItem("points");
localStorage.removeItem("history");

→ 새로고침하면 정상 수치로 표시됨.
(history를 삭제하기 싫다면 points만 삭제 가능)
